JInternalFrames JDialogs y controles para usos generales del sistema

Proporciona clases que crean ventanas internas, jdialogs o controles, que heredan de JPanel, para diversas tareas generales del sistema como: manejo de conexion con el servidor, manejo de datos del docente, ventana de ayuda, ventana acerca de, preferencias del sistema y la ventana principal del sistema.
